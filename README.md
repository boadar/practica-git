# practica-git

Este es un repositorio de práctica para aprender el flujo de trabajo con Git y GitHub.

## Objetivo

El objetivo de este proyecto es entender como funciona un Pull Request de principio a fin:

1. Clonar un repositorio
2. Crear una rama nueva
3. Hacer un cambio pequeño
4. Enviar el cambio con un commit
5. Abrir un Pull Request para revisión

¡Bienvenido a tu primer PR!
